Botões

Botão que abre URL

O parametro para abrir uma nova pagina esta no atributo do form

                    <form action="dummyPage.html">
                        <input type="submit" value="Abrir link">
                    </form>

Botão que chama um script

O botao chama um javaScript para realizar uma ação

                <form>
                    <input type="button" value="Chama script" onclick="botao();">
                </form>
                <!-- Codigo de JavaScript -->
                <script type="text/javascript">
                    function botao()
                    {
                        alert("carregou no botao!");
                    }
                </script>
